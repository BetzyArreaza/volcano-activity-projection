Volcano Activity Projection Model 🌋

This project analyzes volcanic activity patterns of Volcán de Fuego using statistical and data science methods.

Methods Used

  • Linear regression for activity trends
  • Poisson simulation for probabilistic eruption scenarios
  • GOES satellite radiance analysis

Technologies

  • Python
  • Pandas
  • Matplotlib
  • Statistical Modeling
